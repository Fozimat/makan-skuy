# makan-skuy
Aplikasi Katalog Restoran dengan menerapkan :
- PWA
- Tampilan Responsif
- Aksesibilitas
- Integration Test menggunakan Jasmine
- End to End Test menggunakan Codecept
- Image Optimization
- Bundle Optimization

Dummy Restaurant API Reference From [Dicoding](https://restaurant-api.dicoding.dev/)
